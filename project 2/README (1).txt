Golden Spoon Restaurant Landing Page

Overview:
A single-page website showcasing the menu, ambiance, and contact details of a fictional restaurant.

Features:
- Responsive layout using Flexbox
- Hover effects on menu cards
- Font Awesome icons for contact and social media
- Warm color palette for food-themed branding

Structure:
- index.html: Main HTML file
- css/style.css: Styling
- images/food/: Contains food images used in the menu

Optional Enhancements:
- Zoom-in hover effect on menu cards
- CSS-only lightbox (not implemented yet)
