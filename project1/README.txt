Movie Info Page - Inception

This project is a single-page website showcasing the movie "Inception".
It includes a cinematic design, cast grid, user reviews, and streaming info.

Files:
- index.html: Main HTML structure
- css/movie.css: Styling and layout
- images/: Contains movie poster and cast images

Features:
- Responsive layout using Flexbox
- Dark theme with cinematic visuals
- CSS-only star ratings
- Google Fonts for poster-style typography
